這是一個[Next.js](https://nextjs.org/) 項目 bootstrapped 和 [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

##入門

首先，運行開發服務器:

```bash
npm run dev
# or
yarn dev
```

打開 [http://localhost:3000](http://localhost:3000) 使用瀏覽器查看結果.

您可以通過修改 `pages/index.js` 開始編輯頁面。該頁面會在您編輯文件時自動更新.

[API routes](https://nextjs.org/docs/api-routes/introduction) 可以訪問 [http://localhost:3000/api/hello](http://localhost:3000/api/hello). This endpoint can be edited in `pages/api/hello.js`.

`pages/api` 目錄映射到 `/api/*`。此目錄中的文件被視為 [API routes](https://nextjs.org/docs/api-routes/introduction) 而不是 React 頁面.
